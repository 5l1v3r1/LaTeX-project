Elsevier Journals

 

Here you can find templates for Elsevier journals and links to their LaTeX related pages. For getting the templates, please follow the following links:

  1. The homepage for authors
  2. The homepage for LaTeX users
  3. How to prepare LaTeX manuscripts
  4. How to use elsarticle document class
  5. How to make CRC articles

If any link [2-5] is broken, use the home page link. If that is broken as well, then please google for the link with search 'Elsevier LaTeX publishing' and please let the maintainer of the website know about this so that the website could be fixed.

The style files [2] and the support package [4] should be already available in your LaTeX distribution, so at first try downloading only the instructions and sample files [3]. For inpatient the achemso [5] package should do all the necessary things, but please read the documentation before using it. Also, if not sure about the version, please check for updates at the home page[1].

There are copies of the necessary files on this repository and the links bellow are provided just for your own convenience. If you have any problems, check, whether everything is up to date. If you are still having problems afterwards, please refer to the website-links above.
