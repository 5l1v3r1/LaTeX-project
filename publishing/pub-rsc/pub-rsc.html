Royal Society of Chemistry Journals

 

Here you can find templates for all RSC journals. The journals are supported by means of their official templates. For getting the templates, please follow the following links:

  1. The RSC LaTeX template homepage
  2. Template for Communications
  3. Template for Full Papers
  4. Template for Faraday
  5. The BibTeX styles on CTAN

Actually BibTeX styles [5] are in a package called rsc, which should already be in your LaTeX distribution.

If any link [2-5] is broken, use the home page link[1]. If that is broken as well, then please google for the link with search 'LaTeX for RSC journals' and please let the maintainer of the website know about this so that he/she could update the links.

Also, if not sure about the version, please check for updates at the home page[1].

There are copies of the necessary files on this repository and the links bellow are provided just for your own convenience. If you have any problems, check, whether everything is up to date. If you are still having problems afterwards, please refer to the website-links above.
